CACコンテスト「繋がる」で使用するリポジトリ

### 動作環境
Unity バージョン 6000.0.68f1(LTS)

### ゲーム名
[Link to Goal]()

### ゲーム概要
ゴールに「繋がる」道を作って、ゴールを目指すアクションゲーム

### 操作方法
* 移動: A(←)/D(→)
* ジャンプ: Space
* リスタート: R
* 道の設置: 左クリックしながらドラッグ&ドロップ

### 工夫した点
* BaseButton.csというボタンのアニメーションをまとめた抽象クラスを作成し、それを継承されて各ボタンの処理を書いたこと
* シーン遷移するinterfaceを実装したこと
* 可能な限りスクリプトを分割するようにしたこと
